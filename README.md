# TextUtils

Welcome to TextUtils, your go-to platform for text manipulation tools! This website offers a variety of utilities to help you manipulate and analyze text efficiently.

## Overview

TextUtils provides a collection of tools designed to assist users with various text-related tasks, from formatting to analyzing text data. Whether you need to convert text formats, or count characters, TextUtils has you covered.

## Features

- **Text Formatting**: Easily format text with options for converting case (uppercase, lowercase, title case), removing extra spaces, and more.
  
- **Character Count**: Count characters, words, sentences, and paragraphs in your text input.
  
- **Text Analysis**: Analyze text complexity with features like readability score calculation and keyword frequency analysis.

- **Responsive Design**: Ensures the website looks great and functions well across different devices and screen sizes.

## Technologies Used

- HTML5
- CSS3 (Flexbox, Media Queries)
- Tailwind css with Vite
- React js

## Usage

1. Visit the TextUtilities website.
2. Choose the tool you want to use from the navigation menu.
3. Input your text or select options as required by the tool.
4. Use the tool results as needed or download outputs if applicable.
